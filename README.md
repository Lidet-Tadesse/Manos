# Manos

Manos is a charity management system designed to connect donors and recipients through clear, role-based workflows. The platform focuses on transparency, usability, and real-world charity operations.

## Overview
Manos enables donors to contribute and track their donations while allowing recipients to request and receive support efficiently. The system maintains donor history and donation records to ensure accountability and smooth coordination.

## Features
- Separate workflows for donors and recipients
- Donation tracking and donor history management
- Clean and responsive user interface
- Simple and practical system design

## Tech Stack
- Backend: Spring Boot
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

## Project Purpose
This project was developed as a practical, real-world system to apply software engineering concepts such as backend integration, database design, and user-centered UI/UX development.

## Status
The project is functional and actively used as a portfolio project.
